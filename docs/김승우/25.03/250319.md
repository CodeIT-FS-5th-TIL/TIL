
## 배운 내용


### 데이터 모델링

- 개념적 구조 정하는 것  - 논리적 모델링
- DB 구축에 필요한 걸 정하는 것 - 물리적 모델링
- 앤티티(개체) - 실제 대상 하나
	- 학교 관리 프로그램 - 학생, 수업, 교수 등
- 어트리뷰트(속성) - 테이블에서 컬럼
- 관계 - 엔티티들관의 관계
- 제약 조건

### ESLint

- 코드의 퀄리티를 보장해 주기 위한 툴
- ECMAScript + Lint, 즉 JS, TS만을 위함.
- npm으로 설치를 하고, vscode에서도 extension으로 설치를 해야 잘 동작 함.
